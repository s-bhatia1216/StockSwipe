## What It Does

### Discover
- Swipe right to invest, left to skip — physics-based gestures with live BUY/SKIP overlays.
- Flip each card:
  - **Front:** live price, multi-range interactive chart, ticker details.
  - **Back:** company thesis, bull/bear framing, live news widgets, Ask Claude search bar.
- "Why this stock?" hook generated by Claude Haiku per ticker.
- Smart deck ordering learns from swipe history and reorders upcoming cards.

### Reels
- Vertical short-video feed for stocks.
- Tap the stock chip on a reel to open the buy modal directly.

### Portfolio
- Persistent holdings with live pricing updates.
- Swipe each holding right to buy more, left to sell.
- Buy/sell detail modal with share math, MAX sell, and confirmation animation.
- Diversification summary and streak/badge gamification.

### AI Insights
- Claude Sonnet portfolio analysis (grade, headline, summary, strengths, risks, recommendations).
- Risk Radar visualization (concentration, volatility, correlation) with AI explanation.
- Flippable insight cards with plain-English explanations.
- Recommendation cards support:
  - Tap to flip for explanation.
  - Swipe right to open buy flow.
  - Swipe left to dismiss.

### Social Layer
- Friends feed and leaderboard views.
- Community comments with likes, replies, and stance filtering.

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Configure environment

```bash
cp .env.example .env
```

Required variables:

| Variable | Used For |
|----------|----------|
| `ANTHROPIC_API_KEY` | Claude hooks, Q&A, and portfolio insights |
| `FINNHUB_API_KEY` | Company news feed |
| `VITE_YOUTUBE_API_KEY` | Reels feed in frontend |

### 3. Run locally

```bash
npm run dev
```

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:3001`

### 4. Build for production

```bash
npm run build
```

## API Surface

| Endpoint | Description |
|----------|-------------|
| `GET /api/stock/:ticker/:range` | Live price + chart data |
| `GET /api/news-insights/:ticker` | News feed for a ticker |
| `GET /api/hook/:ticker` | Claude-generated "why this stock?" hook |
| `POST /api/ask` | Ask Claude about a stock |
| `POST /api/insights` | Full portfolio AI analysis |
