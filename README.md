# StockSwipe
A Tinder-style stock discovery and AI investing app. Swipe right to invest, swipe left to skip, flip cards for live AI analysis and real news. StockSwipe is a mobile-first investing experience that combines swipe-based stock discovery, live market/news data, and Claude-powered portfolio intelligence.

Built for the AI@Princeton x Trade[XYZ] Hackathon.

## What It Does

### Discover
- Swipe right to invest, left to skip, with physics-based gestures and live BUY/SKIP overlays.
- Flip each card:
  - Front: live price, multi-range interactive chart, ticker details.
  - Back: company thesis, bull/bear framing, live news widgets, Ask Claude search bar.
- "Why this stock?" hook generated by Claude Haiku per ticker.
- Smart deck ordering learns from swipe history and reorders upcoming cards.

### Reels
- Vertical short-video feed for stocks.
- Tap the stock chip on a reel to open the buy modal directly.

### Portfolio
- Persistent holdings with live pricing updates.
- Swipe each holding right to buy more, left to sell.
- Buy/sell detail modal with share math, MAX sell, and confirmation animation.
- Diversification summary and streak/badge gamification.

### AI Insights
- Claude Sonnet portfolio analysis (grade, headline, summary, strengths, risks, recommendations).
- Risk Radar visualization (concentration, volatility, correlation) with AI explanation.
- Flippable insight cards with plain-English explanations.
- Recommendation cards support:
  - Tap to flip for explanation.
  - Swipe right to open buy flow.
  - Swipe left to dismiss.

### Social Layer
- Friends feed and leaderboard views.
- Community comments with likes, replies, and stance filtering.

## Tech Stack

| Layer | Tech |
|-------|------|
| Frontend | React 18, Vite |
| Animation | Framer Motion |
| Charts | Custom Canvas rendering |
| Backend | Express (Node.js) |
| Market Data | yahoo-finance2 |
| News | Finnhub API + og:image extraction |
| AI | Anthropic SDK (Claude Haiku + Sonnet) |
| Video | YouTube Data API v3 |

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Configure environment

Create a `.env` file in the project root:

```bash
cp .env.example .env
```

Required variables:

| Variable | Used For |
|----------|----------|
| `ANTHROPIC_API_KEY` | Claude hooks, Q&A, and portfolio insights |
| `FINNHUB_API_KEY` | Company news feed |
| `VITE_YOUTUBE_API_KEY` | Reels feed in frontend |

### 3. Run locally

```bash
npm run dev
```

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:3001`

### 4. Build for production

```bash
npm run build
```

## API Surface

Implemented backend endpoints:

- `GET /api/stock/:ticker/:range`
- `GET /api/news-insights/:ticker`
- `GET /api/hook/:ticker`
- `POST /api/ask`
- `POST /api/insights`

## Scripts

- `npm run dev` - run frontend + backend concurrently
- `npm run server` - run backend only
- `npm run build` - production build
- `npm run preview` - preview production build
